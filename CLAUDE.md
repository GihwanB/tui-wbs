# 응답 규칙
- 한국어 기본 언어 사용
- 결과 출력 시 다음 형식 준수:
  - **질의**: 사용자가 요청한 내용 요약 (1줄)
  - **해석**: 질의에 대한 간단한 해석 또는 접근 방향 (1~2줄)
  - 이후 실제 결과 출력

# Testing

- TUI 앱 테스트 시 tmux 세션을 사용한다:
  ```bash
  tmux attach -t tui-wbs || tmux new -s tui-wbs
  ```
- 특별한 이유가 없으면 tmux 세션(`tui-wbs`)은 종료하지 않는다.
- tmux 명령어에 `2>&1` 같은 출력 리다이렉션(`>`)을 사용하지 않는다.
- 복합 명령어(`;`, `|` 체인) 대신 개별 Bash 호출로 분리한다.
